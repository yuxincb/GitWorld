---本地---

初始化版本库
git init

把当前目录下所有文件添加到暂存区
git add .

把暂存区文件提交到当前分支
git commit -m "init the repository"

查看git状态
git status

查看文件差异
git diff hello.txt

查看版本库和工作区文件差异
git diff HEAD -- hello.txt

版本历史
git log

单行显示版本历史
git log --pretty=oneline

操作历史
git reflog

回滚到上一版本
git reset --hard HEAD^

回滚到指定版本
git reset --hard 650d

丢弃工作区修改
git checkout -- hello.txt

---远程---

与Github远程库关联
git remote add origin git@github.com:yuxincb/GitWorld.git

第一次推送master分支的所有内容
git push -u origin master

推送修改
git push origin master

从远程克隆一个本地库
git clone git@github.com:yuxincb/GitWorld.git

---分支---

创建并切换到dev分支
git switch -c dev
git checkout -b dev

创建dev分支
git branch dev

切换到dev分支
git switch dev
git checkout dev

查看当前分支
git branch

把dev分支合并到当前分支
git merge dev

删除dev分支
git branch -d dev
